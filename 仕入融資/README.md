# 仕入融資アプリ ダッシュボード機能

kintoneの「仕入融資」アプリにおいて、銀行別の利息集計や外部金利（TIBOR）の同期を一括で行うための高度なJavaScriptカスタマイズです。

## 機能概要
1. **TIBOR金利同期 (`bank-dashboard.js`)**
   - 外部の金利管理アプリから最新の1ヶ月・3ヶ月TIBORを取得し、現在のビューに表示されている全レコード（上限なし・バッチ処理）へワンクリックで反映させます。
2. **銀行別 利息累計ダッシュボード**
   - 確定返済日が翌月以降、または未記入のレコードを抽出し、銀行ごとの合計利息額を算出。
   - 降順ソートされた集計テーブルを一覧画面上部にアコーディオン形式で表示します。
3. **柔軟な権限制御**
   - 特定のユーザー名、またはロール（システム管理者等）を持つユーザーにのみダッシュボードを表示する制御機能を搭載。

## 技術仕様
- **ファイル**: `bank-dashboard.js`
- **対象イベント**: `app.record.index.show` (一覧表示時)
- **API利用**: `fetchAllRecords` (カーソル不要の全件取得ロジック), `kintone.api.url('/k/v1/records')`

## AIへの改修指示用メモ
- 更新ボタンは一度に100件ずつのバッチ更新を行うため、大量のレコードがあってもタイムアウトを回避します。
- UIのデザインは `app.record.index.show` 内で直接挿入されており、`sessionStorage` を用いてアコーディオンの開閉状態を記憶します。
