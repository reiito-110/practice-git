# マニュアル自動作成ツール

PowerPointファイル（.pptx）を解析し、スライドのレイアウトを再現したHTMLファイルへと自動変換するPythonツールです。

## 機能概要
- **PPTX解析**: スライド内のテキスト、画像、図形の位置関係をEMU単位で取得。
- **HTML/CSS生成**: 解析したレイアウトを絶対配置（Absolute positioning）を用いたHTMLとモダンなCSSで再現し、ブラウザで閲覧可能なマニュアルを作成します。
- **資産抽出**: スライド内の画像ファイルを `.png` 等の形式で自動抽出し、アセットとして保存します。

## 技術仕様
- **ファイル**: `convert_pptx.py`, `manual_correction.py`
- **言語**: Python (python-pptx)
- **出力**: `manual_output/` フォルダ内にHTMLと画像一式。

## AIへの改修指示用メモ
- CSSのスタイル（フォントサイズや背景色など）は `convert_pptx.py` 内の `extract_shape` 関数で調整可能です。
- 現時点ではグループ化された図形の完全な再現には制限がありますが、フラットな要素として出力されます。
