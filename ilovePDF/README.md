# kintone ilovePDF連携プラグイン

kintone内のデータをPDF操作ツール等と連携、またはカスタマイズするためのプラグインソースです。

## 機能概要
- **PDF連携/操作**: kintoneアプリ上でのPDF生成や加工に関連する機能を提供します。
- **プラグイン形式**: そのままパッケージングしてkintoneに導入可能な構成になっています。

## 技術仕様
- **主要フォルダ**: `src/` (manifest.json, js, html, css など)
- **開発言語**: JavaScript, CSS, HTML

## AIへの改修指示用メモ
- プラグインの設定画面（Config）と実行画面（Desktop）のロジックが分離されています。
- マニフェストファイル（`manifest.json`）の定義に従って動作するため、ファイル追加時はマニフェストの更新を忘れないでください。
