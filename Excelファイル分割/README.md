# Excelファイル分割ツール

大量の行を含むCSVファイルを指定した行数（デフォルト100行）ごとに分割し、Excel形式（.xlsx）で保存・GoogleドライブへアップロードするPythonスクリプトです。

## 機能概要
- **CSV分割**: ドラッグ＆ドロップされたCSVファイルを読み込み、一定行数ごとに分割したExcelファイルを生成します。
- **自動カテゴリ分類**: ファイル名からカテゴリを抽出し、対応するフォルダに自動仕分けします。
- **Googleドライブ連携**: 生成したファイルをGoogleドライブ（Gドライブ）の `AUBA_Split` フォルダへ `robocopy` を使用して自動アップロードします。

## 技術仕様
- **ファイル**: `split_csv.py`
- **言語**: Python (pandas, openpyxl)
- **連携**: Google Drive for Desktop (Gドライブ)

## AIへの改修指示用メモ
- 分割行数は `LINES_PER_FILE` 定数で変更可能です。
- アップロード先は `G:\マイドライブ\AUBA_Split` を優先的に探索します。
