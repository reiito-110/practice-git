# 車両管理アプリ カスタマイズ

kintoneの車両管理アプリにおいて、外部からの遷移時に車両情報を自動入力するJavaScriptカスタマイズです。

## 機能概要
- **車両情報の自動セット**: URLパラメータ `car` が指定されている場合、レコード追加画面の表示時に「車両」ドロップダウンフィールドへその値を自動的にセットします。

## 技術仕様
- **ファイル**: `desktop.js`
- **対象イベント**: `app.record.create.show`
- **パラメータ**: `?car=車両名` (URLエンコードが必要)

## AIへの改修指示用メモ
- このプログラムはドロップダウン項目の値を書き換えます。選択肢にない値をパラメータで渡すとエラーになるため、マスターデータとの整合性に注意してください。
