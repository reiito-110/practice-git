# アフターサービス保証書 カスタマイズ

kintoneのアフターサービス保証書アプリと仕入融資アプリの間で、データを同期・連携させるためのJavaScriptカスタマイズです。

## 機能概要
- **物件番号の同期とルックアップ自動実行**
  - 「物件番号」フィールドが変更された際、その値を「物件番号_仕入融資アプリ」（ルックアップフィールド）へ自動コピーし、ルックアップを実行します。
  - 編集画面表示時にも、値が空であれば自動補完を試みます。

## 技術仕様
- **ファイル**: `kintone-field-sync.js`
- **対象イベント**: `app.record.create.change.物件番号`, `app.record.edit.change.物件番号`, `app.record.edit.show`

## AIへの改修指示用メモ
- このスクリプトは他のアプリ（仕入融資アプリ）とのルックアップ連携を前提としています。フィールドコードが変更された場合は、冒頭の定数を修正してください。
