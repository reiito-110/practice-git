# 貯蔵品在庫管理 カスタマイズ

kintoneの貯蔵品在庫管理アプリにおいて、入力漏れ防止と入力作業の効率化を行うためのJavaScriptカスタマイズです。

## 機能概要
1. **プロセス管理 入力漏れ防止 (`process_alert.js`)**
   - アクション「管理者承認」実行時に確認ダイアログを表示。
   - 「実際在高タブ」の入力を促し、キャンセル時はステータス更新を中断します。
2. **所属組織の自動補完 (`user_org_autofill.js`)**
   - ユーザー選択フィールド（レターパック、印紙、切手）が変更された際、そのユーザーの所属組織を自動取得してセットします。
   - 役職名や特定の課（契約事務課など）を除外するフィルタリングロジックを搭載しています。

## 技術仕様
- **ファイル**: `process_alert.js`, `user_org_autofill.js`
- **対象イベント**: `app.record.detail.process.proceed`, `app.record.create.change.*`, `app.record.edit.change.*`
- **API利用**: `kintone.api('/v1/user/organizations')` を使用してユーザーの組織情報を取得。

## AIへの改修指示用メモ
- 組織のフィルタリングロジックは `EXCLUDED_KEYWORDS` 定数で管理されています。除外対象を増やしたい場合はここを修正してください。
- テーブル形式の入力履歴に対応しており、変更された行を特定して自動セットを行います。
