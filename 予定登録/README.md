# 予定登録・マニュアル管理

kintoneの「予定登録」アプリに関連するマニュアル資料と、その保守・検証用の補助スクリプト群です。

## 機能概要
- **マニュアル資料**: 予定アプリの使い方を解説したHTML、PDF、およびPPTX形式のユーザーマニュアル。
- **補助スクリプト**: マニュアル作成や内容抽出を効率化するためのPythonスクリプト（試作版）が含まれています。
- **UIパーツ**: 予定登録ボタンのアイコン画像など、アプリカスタマイズに必要な素材。

## 構成
- **ドキュメント**: `scheduled_app_manual.html`, `20250915_予定アプリマニュアル_Ver1.pdf`
- **スクリプト**: `extract_pptx_temp.py`, `peek_pdf_temp.py` (検証用)

## AIへの改修指示用メモ
- マニュアルを更新する場合は、元のPPTXを編集するか、生成済みのHTML/CSSを直接修正してください。
- `kintone_circular_text.txt` には、アプリ内で利用する定型文などが含まれています。
