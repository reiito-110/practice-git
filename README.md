# Kintone カスタマイズ・プロジェクト

このリポジトリは、KintoneアプリのJavaScriptカスタマイズ・ソースコードを管理するためのものです。

## 管理アプリ一覧

### 1. 販売取引報告アプリ
一括ファイルダウンロード機能を提供します。

- **ファイルパス**: `apps/販売取引報告/BusinessPlan_File_Download.js`
- **機能概要**: 
  - 特定の一覧（経理課確認）において、画面上に表示されている全レコードの添付ファイルを一括ダウンロードします。
  - 大量ダウンロードによるブラウザ負荷を考慮し、順次処理（非同期制御）を行っています。
- **設定が必要な箇所**:
  - `VIEW_NAME`: 対象とする一覧画面の名前。
  - `FILE_FIELD`: 添付ファイルフィールドのフィールドコード。

## 開発上の注意点
- kintone.proxy はバイナリデータ破損の恐れがあるため、直接 XMLHttpRequest を使用してファイルを Blob 形式で取得しています。
- 多重描画防止のため、ボタンの二重生成チェックを実装しています。
