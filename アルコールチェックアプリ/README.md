# アルコールチェックアプリ カスタマイズ

kintoneのアルコールチェックアプリにおいて、モバイル利用を最適化するためのJavaScriptカスタマイズです。

## 機能概要
1. **モバイル用パラメータ取得とルックアップ自動実行**
   - URLパラメータ `car` を取得し、「ナンバー」フィールドにセット。
   - その後、取得した値でルックアップ（Lookup）を自動的に実行します。
2. **報告種別に応じたフィールド表示制御**
   - 「報告種別」が **運転前** の場合：`運転終了時メーター距離` を非表示。
   - 「報告種別」が **運転後** の場合：`行先`、`運転開始時メーター距離`、`日常点検` グループを非表示。

## 技術仕様
- **ファイル**: `mobile.js` (モバイル版カスタマイズ)
- **対象イベント**: `mobile.app.record.create.show`, `mobile.app.record.create.change.報告種別` など
- **パラメータ取得ロジック**: 正規表現を用いて、クエリ文字列（?）、フラグメント（#）の両方から安全に値を取得します。

## AIへの改修指示用メモ
- 表示制御は `kintone.mobile.app.record.setFieldShown` を使用しています。PC版には適用されません。
- ルックアップの自動実行が含まれているため、マッピング先のフィールド設定が正しいことを前提としています。
