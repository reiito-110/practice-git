# kintone フィールド権限/表示制限プラグイン

kintoneの標準機能では難しい、動的なフィールドの表示・非表示制御や編集制限を実現するプラグインです。

## 機能概要
- **フィールド制御**: 条件に基づいてフィールドを表示・非表示にしたり、読み取り専用に設定します。
- **設定UI**: 管理画面からプログラミングなしで制御ルールを構成可能です。

## 技術仕様
- **主要フォルダ**: `js/`, `html/`, `css/`
- **言語**: JavaScript
- **ロジック**: `desktop.js` で実行時の制御、`config.js` で設定画面を構築。

## AIへの改修指示用メモ
- 制御対象のフィールドコードや条件式は、プラグイン設定（Config）から渡される JSON データに基づいています。
- `kintone.app.record.setFieldShown` を利用しています。
